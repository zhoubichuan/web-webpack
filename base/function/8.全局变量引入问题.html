# 十.全局变量引入

jquery 引入的几种方式（在处理 js 语法的基础上改）

  • 1） expose-loader 暴露到 window 上
  • 2）providePlugin 给每个模块提供一个$
  • 3）引入不打包

# 1.安装

cnpm i jquery -S
1
cnpm i expose-loader -D
1

# 2.文件

src/index.js 添加

import $ from "expose-loader?$!jquery";
console.log($);
console.log(window.$);
